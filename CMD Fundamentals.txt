Check version:
	git --version

Set config values:
	git config --global user.name "Khalil Ahmed"
	git config --global user.email "mkhalila@outlook.com"

View config values:
	git config --list

Help:
	git help <verb> (e.g. git help config)
	git <verb> --help (e.g. git add --help)

Initialise a Repo from Existing code:
	git init

Link local repo to remote repo (Initial - after creating repo on github):
	git remote add origin https://github.com/mkhalila/<remote repo name>.git
	git push -u origin master

Status of repo:
	git status

Add gitignore file to local repo
	touch .gitignore

Add files to staging area
	git add -A
	git status

Commit changes:
	git add -A
	git commit -m "<Commit message here>"
	git status

Cloning a remote repo:
	git clone <url> <where to clone> ("." = current directory)
	E.g. clone https://github.com/mkhalila/Git-Tutorials.git .

View branches of repo:
	git branch -a

Pushing changes:
	(Commit):
		git diff
		git status
		git add -A
		git commit -m "<Commit message>"
	(Push):
		git pull origin master
		git push origin master

Create a branch:
	git branch <branch-name>

Checkout a branch:
	git checkout <branch-name>

After commit, push entire new branch to remote: (First time after creating branch)
	git push -u origin <branch-name>
	git branch -a

Merge a branch with master:
	git checkout master
	git pull origin master
	git branch --merged
	git merge <branch-name>
	git push origin master

Delete a branch (e.g. After merging it with master):
	git branch -d <branch-name> (deletes locally)
	git branch -a 
	git push origin --delete <branch-name> (deletes remote branch)
	
Update a fork with upstream changes
	git remote add upstream git@github.ibm.com:development-enablement/SADashboards.git
	git fetch upstream
	git merge upstream/master
	git push origin
